# Pikku Eläintarha

Pikku Eläintarha is a simple web application that lets users **view, add, remove, search, and filter animals** by type. It's built with **HTML, CSS, and JavaScript** — no backend required.

## Features

- Display a list of animals with their types.
- Add new animals with name and type.
- Remove animals individually.
- Filter animals by type (Nisäkäs, Lintu, Matelija, Sammakkoeläin, Kala).
- Search animals by name (real-time search).
- Sort animals alphabetically (A → Ö).

## Usage
View Animals: The list of animals is displayed by default.
Add Animal: Enter a name, select type, and click "Lisää eläin".
Remove Animal: Click "Poista" next to any animal to remove it.
Search Animals: Type a name in the search box to filter the list.
Filter by Type: Select an animal type from the dropdown to see only that category.
Sort Animals: Click "Järjestä A-Ö" to sort alphabetically.
